# 🚀 ¡Hola y Bienvenido a Mi Sitio Web!  

Este repositorio contiene el código de mi sitio web personal, diseñado para mostrar mi experiencia, proyectos y conocimientos en desarrollo y diseño UX/UI. Aquí encontrarás una vitrina interactiva con información sobre mi carrera profesional y proyectos destacados.  

---

## 💻 **¿Qué encontrarás en este repositorio?**  

- **Sitio Web Personal:** Una experiencia interactiva y atractiva que refleja mi enfoque en diseño y desarrollo.  
- **Proyectos Destacados:** Información sobre mis trabajos en diseño UX/UI y desarrollo iOS.  
- **Diseño Adaptable:** Optimizado para funcionar en cualquier dispositivo (móvil, tablet o escritorio).  
- **CV Descargable:** Acceso directo a mi currículum para conocer más sobre mi experiencia profesional.  

---

## 🛠️ **Tecnologías utilizadas para construir mi web**  

Este sitio está desarrollado utilizando las siguientes tecnologías:  

- **HTML5 & CSS3:** Para crear la estructura y el diseño atractivo.  
- **JavaScript & jQuery:** Para agregar interactividad y una mejor experiencia de usuario.  
- **GSAP:** Animaciones suaves y profesionales.  
- **Pace.js:** Animación de carga con un diseño elegante y minimalista.  

---

## 👀 **¿Cómo visualizar mi sitio web?**  

1. Abre el archivo `index.html` en tu navegador favorito.  
2. Explora las secciones para descubrir información sobre mi experiencia, proyectos y enfoque profesional.

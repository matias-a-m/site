## Tecnologías Utilizadas

Este portafolio se construyó con:

- **HTML5** para estructurar el contenido.
- **CSS3** para darle estilo y hacer que todo luzca atractivo.
- **JavaScript** para añadir interactividad.
- **jQuery** para simplificar la manipulación del DOM.
- **GSAP** para animaciones suaves y profesionales.
- **Pace.js** para mostrar una animación de carga elegante.
